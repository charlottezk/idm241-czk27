1. Home page load

Center "language/idioma" circle starts to fade-out, while language option circles roll out from either side. The circles will land in a legibal position and remain till clicked. No Loops. Modal (including language button, English/Español buttons, x-out button, modal background), blurred background, and page changes are all modes.

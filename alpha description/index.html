Alpha Description

Original Website URL: etsy.com

Trigger

  • Hover state (on/off) & Click

Rules

  • Once curser is hovered over "add to cart" button, curser changes from arrow to hand.
  • In hover-on state, button changes from #222222 to #2F2F2F.
  • Button size instantaniously starts to increase by total of 19%, over 1s. Curve starts steep and gradually gets slower, to create swell affect.
  • Quickly changes back to orginaly size, "swell" animation restarts and loops until "hover-off" state occurs.
  • If curser is hovered off, the state goes back to original state.
